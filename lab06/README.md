## Практические навыки

- Как подходить к решению сложных задач?

Любую сложную задачу можно разбить на несколько более простых задач, такое действие проделываем пока не доходим до элементарных задач, которые супер просто выполнить

- Как список из чисел записать в линейной памяти?

записать числа в массив, где числа будут следовать одно за другим

- Что нужно сделать, чтобы пройтись по всем элементам списка? Посчитать сумму элементов списка?

Нужно выделить ячейку или переменную для текущего эемента списка, сначала она будет навна 0 (индекс 0), после этого увеличивать индекс на 1 переходя к следующему элементу
Аналогично с суммой, в переменную с текущей суммой записываем 0, после к текущей сумме прибавляем значение следующего элемента списка.

создать пустой список для результата result
нужно проверить одинаковая ли длина обоих массивов, если нет выбросить ошибку
создать ячейку памяти под индекс i и записать в нее 0
